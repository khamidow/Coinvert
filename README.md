## ✨ Features

### 💹 Currency Conversion
- Real-time exchange rates
- Convert between multiple currencies
- Instant recalculation while typing
- Accurate and up-to-date market data

### 📊 Historical Rates
- View exchange rates for previous dates
- Select any date using an integrated calendar
- Analyze currency changes over time

### 🌍 Multi-language Support
- English
- O‘zbek
- Ўзбек (Кирилл)
- Русский

### 🎨 Modern User Experience
- Clean Material Design interface
- Smooth animations and transitions
- Swipe-to-refresh support
- Responsive layouts for all screen sizes

### 📱 Cross Platform
- Android support

- Single codebase with Flutter

### ⚡ Performance
- Optimized state management

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/khamidow/Coinvert.git
```

### Navigate to project

```bash
cd Coinvert
```

### Install dependencies

```bash
flutter pub get
```

### Run application

```bash
flutter run
```
